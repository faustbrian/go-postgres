# Roadmap

## v1.0.0

- complete PostgreSQL 14-18 hosted integration evidence
- keep exact meaningful production coverage and all release gates green
- complete API, security, operational, adoption, and compatibility review
- publish the first stable API only after hosted CI confirms the tagged commit

## After v1

Future work may add narrowly scoped pgx hooks or observability instruments.
The project will not become an ORM, query builder, migration engine, repository
generator, transparent retry executor, or generic database abstraction.
